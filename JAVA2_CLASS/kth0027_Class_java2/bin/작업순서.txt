	DDL
		create table DB명.테이블명(
								필드명1 타입 속성,
								필드명2 타입 속성,
								필드명3 타입 속성
		)
		
	* 자료형
			int : 정수			4바이트
			varchar : 문자열		(바이트)
			timestamp : 날짜/시간	4바이트
			
	* 속성 
			primary key : 기본키(PK)
			auto_increment : 자동번호 주입
			not null : null값 제외
			default : 초기값
			
	* 함수
			now() : 현재 날짜/시간

				회원번호**				게시판번호**  --------		댓글번호
				회원아이디----			게시판제목					댓글내용
				회원비밀번호			게시판내용					댓글작성자

				PK : 식별 가능한 키 [ 식별 필드 : 중복X , null X ]
						* 주빈등록번호, 사번, 학번, 카드번호, 사업장번호, 주소 등
						* 다른 테이블과 연결 시 사용되는 식별키 [ 대부분1 ]
						
				FK : PK와 연결된 필드
						* 외래키
						* 다른 테이블의 PK와 연결된 필드 [ 대부분M ]
